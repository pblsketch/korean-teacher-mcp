# 한국 교육 문서 작성 원칙 — AI 생성 텍스트 가이드

> 교육 문서(학습지, 평가지, 공문, 수업 계획서 등)를 생성할 때 참조하는 한국어 작문 원칙입니다.
> AI가 교육 자료 텍스트를 생성할 때 이 원칙을 따르면 현직 교사가 즉시 사용할 수 있는 품질을 확보합니다.

---

## 1. 개조식 작성 원칙

### 1.1 명사형 종결

교육 문서의 기본 문체는 **개조식(bullet-style)**이며 명사형으로 종결합니다.

| 종결 어미 | 사용 맥락 | 예시 |
|-----------|----------|------|
| `-함` | 동작·행위 완료 | 학습 목표 달성 여부 확인함 |
| `-임` | 상태·판단 서술 | 평가 기준은 성취기준 기반임 |
| `-됨` | 피동·완료 상태 | 교과서 3단원과 연계됨 |
| `-요함` / `-필요` | 필요성 서술 | 사전 학습 확인 필요 |

**금지**: 개조식 문서에서 `-합니다`, `-입니다` 혼용 금지. 한 문서 안에서 문체 통일 필수.

### 1.2 한 문장 46자 이내

- 한 항목(bullet)의 텍스트는 **46자(공백 포함) 이내** 작성
- 46자 초과 시 두 항목으로 분리하거나 불필요한 수식어 제거
- 표 셀 내 텍스트도 동일 기준 적용

| 수정 전 (52자) | 수정 후 (38자) |
|---------------|---------------|
| 학생들이 자기주도적으로 학습할 수 있는 환경을 조성하는 것이 중요합니다 | 자기주도 학습 환경 조성 필요 |

### 1.3 주어 생략

문맥상 주어가 명확하면 생략합니다.

| 수정 전 | 수정 후 |
|---------|---------|
| 교사는 학습 목표를 제시한다 | 학습 목표 제시 |
| 학생들은 모둠별로 토의한다 | 모둠별 토의 실시 |
| 교사가 형성평가를 실시한다 | 형성평가 실시 |

### 1.4 불필요한 접미사 제거

| 제거 대상 | 수정 전 | 수정 후 | 규칙 |
|-----------|---------|---------|------|
| `-적` | 효과적인 학습 | 효과 있는 학습 | 한자어+적 → 고유어 표현 |
| `-적` | 창의적 사고 | 창의 사고 | 이미 명사로 충분한 경우 |
| `-의` (연쇄) | 학생들의 학습의 목표 | 학생 학습 목표 | 조사 `-의` 2회 이상 연속 금지 |
| `-것` | 학습하는 것 | 학습 | 명사화 가능 시 제거 |
| `-들` | 학생들, 자료들 | 학생, 자료 | 복수 의미 문맥상 명확 시 생략 |
| `-에 대해` | 주제에 대해 토의 | 주제 토의 | 직접 목적어로 전환 |

## 2. 문서 유형별 가이드

### 2.1 학습지/워크시트

**지시문 작성법**
- 동사 명령형 사용: `~하시오`, `~쓰시오`, `~찾아보자`
- 한 지시문에 **행동 동사 1개**만 포함 (복합 지시 금지)
- 난이도 순서 배치: 쉬운 문제 → 어려운 문제

| 수정 전 | 수정 후 |
|---------|---------|
| 다음 글을 읽고 물음에 답하시오 | 다음 글을 읽고 물음에 답하시오. |
| 아래 보기에서 알맞은 것을 골라 빈칸에 쓰고 그 이유를 설명하시오 | (1) 보기에서 알맞은 말을 골라 빈칸에 쓰시오. (2) 그렇게 고른 이유를 쓰시오. |

**보기/예시 제시법**
- 보기는 `[보기]` 또는 `<보기>` 표기로 통일
- 예시는 `(예)` 또는 `[예시]`로 명시
- 보기 항목은 `ㄱ, ㄴ, ㄷ` 또는 `①, ②, ③` 사용
- 본문과 보기 사이 한 줄 공백 유지

### 2.2 평가지

**문항 작성법**
- 발문(질문)은 **긍정형** 우선 (`옳은 것은?` > `옳지 않은 것은?`)
- 부정형 발문 시 **밑줄+굵게** 강조: `옳지 **않은** 것은?`
- 발문 끝에 물음표 사용 통일
- 한 문항에 측정 요소 1개만 포함

**선지(선택지) 구조 통일**
- 선지 길이 균등 (최장 선지와 최단 선지 차이 10자 이내)
- 선지 번호: `①, ②, ③, ④, ⑤` 통일
- 선지 문법 구조 통일 (모두 명사형 또는 모두 서술형)
- `모두 맞다`, `없다` 등 포괄 선지 지양

**정답·해설·채점 기준**
- 학생용 문항과 교사용 정답·해설은 영역을 분리
- 선택형 해설은 정답 근거와 오답이 틀린 이유를 1문장씩 제시
- 서술형은 모범 답안, 인정 답안, 부분 점수 기준을 함께 제시
- 문항별 배점 합계와 총점은 반드시 일치
- 형성평가는 5문항 안팎으로 핵심 개념·적용·성찰을 균형 있게 구성
- 평가 목표는 `공감적 대화 방법을 실제 갈등 상황에 적용할 수 있다`처럼 관찰 가능한 행동으로 작성

| 나쁜 선지 구조 | 좋은 선지 구조 |
|---------------|---------------|
| ① 사회적 배경 ② 인물이 성장하는 과정 ③ 갈등 ④ 반어법을 사용함 | ① 사회적 배경 ② 인물의 성장 ③ 인물 간 갈등 ④ 반어적 표현 |

### 2.3 공문/통신문

**필수 규칙**
- 개조식 **필수** (서술형 문장 사용 금지)
- 경어체 유지 (`-하겠습니다`, `-바랍니다`, `-알려 드립니다`)
- 결재 문서 형식 준수: 수신, 참조, 제목, 본문, 끝 순서
- 제목: 20자 이내, 핵심 키워드 선두 배치
- 본문 `1. → 가. → 1) → 가)` 계층 구조 사용

**개조식 계층 구조**
```
1. 대항목 (아라비아 숫자 + 마침표)
  가. 중항목 (한글 가나다 + 마침표)
    1) 소항목 (아라비아 숫자 + 닫는 괄호)
      가) 세부항목 (한글 가나다 + 닫는 괄호)
```

### 2.4 수업 계획서/지도안

**학습 목표 기술법**
- **ABCD 구조** 활용: 대상(Audience) + 행동(Behavior) + 조건(Condition) + 정도(Degree)
- 행동 동사는 관찰 가능한 것 사용 (블룸 택소노미 기반)
- `~을 이해한다` → `~을 설명할 수 있다` (측정 가능하게 전환)

| 수준 | 피해야 할 동사 | 권장 동사 |
|------|--------------|----------|
| 기억 | 안다, 이해한다 | 열거한다, 나열한다 |
| 이해 | 파악한다 | 설명한다, 비교한다 |
| 적용 | 활용한다 | 적용한다, 해결한다 |
| 분석 | 생각한다 | 분석한다, 분류한다 |
| 평가 | 느낀다 | 판단한다, 평가한다 |
| 창조 | 배운다 | 설계한다, 구성한다 |

**지도안 필수 구조**
- 수업 정보: 학년, 과목, 단원, 차시, 수업자, 장소
- 성취기준: 코드와 문장 함께 기재
- 학습 목표: 학생 행동 중심 2~3개
- 수업 흐름: 도입-전개-정리 단계별 시간 배분
- 활동 구분: 교사 활동과 학생 활동 분리
- 자료/유의점: 판서, 활동지, 디지털 자료, 지도상 유의점 명시
- 평가 계획: 평가 관점, 방법, 피드백 방식 포함

**활동 단계 서술**
- 도입(5~10분) → 전개(25~30분) → 정리(5~10분) 3단계 표기
- 각 단계에 **예상 소요 시간** 필수 기재
- 교사 활동과 학생 활동 **분리 기술**
- 평가 관점은 성취기준 코드와 연계 명시

### 2.4-1 형성평가지와 수행평가지 구분

**형성평가지(assessment 템플릿) 필수 구조**
- 목적: 수업 중·후 핵심 개념 이해와 적용 여부를 빠르게 확인
- 구성: 학생용 문제지 + 교사용 정답 및 해설지
- 학생용 문제지: 상단 정보, 안내문, 문항, 배점, 답란만 포함
- 선택형 답란은 긴 줄답란 대신 `정답: (____)` 형태로 두고, 서술형 문항에만 충분한 줄 답란을 제공
- 학생용 문제지 끝에는 `- 끝 -`을 넣어 시험지형 문서로 마무리
- 교사용 정답 및 해설지: 문항별 정답, 해설, 채점 기준, 배점, 난이도, 평가 요소 포함
- 정답·해설은 학생용 문제지와 반드시 별도 쪽으로 분리해 인쇄/배부 시 제외 가능하게 작성
- 형성평가는 5문항 안팎으로 핵심 개념, 보기 적용, 짧은 서술형 성찰을 균형 있게 구성

**수행평가지(performance-assessment 템플릿) 필수 구조**
- 목적: 학생이 일정 시간 동안 수행한 과정·산출물·성찰을 평가
- 구성: 수행 과제 안내, 성취기준/평가 목표, 수행 절차, 제출물, 유의사항, 평가 기준표, 실제 학생 활동지, 자기 성찰
- 문항 풀이식 문제보다 과제 상황과 산출물 조건을 먼저 제시
- 평가 기준표는 평가 영역, 배점, 우수/보통/미흡 수준을 구체적 행동 증거로 작성
- 학생이 무엇을 제출해야 하는지, 언제까지, 어떤 형식으로 제출하는지 명확히 제시
- 제출 분량, 제출 형식, 개인정보 익명 처리, 허위 작성/표절 시 처리, 필수 포함 조건을 별도 유의사항으로 제시
- 수행평가 문서 안에는 학생이 실제로 작성할 활동지를 포함해야 하며, 활동 자료·활동 문항·학생 작성란을 분리
- 자기 성찰란은 최소 4문항으로 구성해 잘한 점, 어려웠던 점, 개선 계획, 배운 점을 기록하게 함
- 채점 기준은 교사용 비밀 답안이 아니라 학생에게 공개 가능한 성공 기준으로 작성

**공통 문항/기준 작성법**
- 발문은 긍정형 우선 사용
- 한 문항 또는 평가 영역에 측정 요소 1개만 포함
- 선택지는 길이와 문법 구조를 균일하게 작성
- 서술형은 답안 조건과 채점 요소를 명시
- 수행평가 루브릭은 `열심히 참여함` 같은 태도 일반론보다 관찰 가능한 산출물·발화·수정 과정으로 기술

### 2.4-2 PBL/토의토론 자료

**PBL 설계서 구조**
- 프로젝트 제목과 핵심 질문(Driving Question) 제시
- 실제성 있는 문제 상황과 최종 산출물 명시
- 차시별 활동, 자료, 교사 지원, 평가 계획 포함
- 공개 발표/공유 방식과 성찰 활동 포함

**토의토론 자료 구조**
- 논제, 배경 자료, 핵심 용어 정의 먼저 제시
- 입장 정리표, 근거 찾기표, 발언 기록표 포함
- 토론 규칙과 역할 분담 명시
- 토론 후 성찰 및 자기/동료 평가 포함

### 2.5-1 공문·회의록·보고서

**공문**
- 수신, 경유, 제목, 본문, 붙임, 끝 순서 유지
- 내부결재와 외부 발송 공문을 구분
- 외부 공문은 시행번호, 시행일자, 주소, 전화, 팩스, 이메일 포함
- 본문은 `1.`, `가.`, `1)`, `가)` 계층 사용
- 감정적 표현과 구어체 금지

**회의록**
- 회의명, 일시, 장소, 참석자, 작성자 먼저 제시
- 안건별로 논의 내용과 결정 사항을 분리
- 후속 조치는 담당자와 기한을 함께 기록
- 발언 전문보다 결정 가능한 요약문으로 정리
- 개인정보·민감 발언은 필요 최소한으로 익명화

**보고서**
- 문서 번호, 작성일, 작성자, 부서 등 메타 정보 포함
- 목적/배경 → 주요 내용 → 분석/결과 → 제언 순서 권장
- 사실, 해석, 제안을 구분하여 작성
- 표·그래프 설명은 본문에서 핵심 해석을 먼저 제시

### 2.5-2 계획서·체크리스트·명렬표

**운영계획서/제안서**
- 목적, 방침, 추진 내용, 일정, 역할, 예산, 기대 효과 포함
- 모든 추진 항목에 담당자와 기한을 연결
- 예산은 산출 근거와 합계를 함께 제시
- 기대 효과는 추상적 표현보다 관찰 가능한 변화로 작성

**체크리스트**
- 항목은 완료 여부를 판단할 수 있는 동사형으로 작성
- 기한, 담당자, 완료 표시, 비고 열 포함
- 필수/선택 항목을 구분
- 행사·평가·안전 등 유형에 맞는 점검 항목만 포함

**명렬표/자리배치표**
- 학생 번호와 이름을 기본 열로 두고 목적별 열을 추가
- 출석부, 상담기록, 연락처, 과제 확인표를 구분
- 개인정보가 포함될 경우 보관 목적과 접근 범위 제한
- 인쇄 시 기록 가능한 셀 크기 확보

### 2.6 가정통신문

**학부모 대상 문체**
- 존칭 사용: `학부모님께`, `가정에서`
- 전문 용어 사용 시 괄호 안에 쉬운 설명 병기
- 문장형과 개조식 혼용 가능 (안내 부분은 서술, 세부 사항은 개조식)
- 계절 인사 → 안내 배경 → 협조 요청 순서로 자연스럽게 전개
- 사용자가 입력한 학교명·부서명·날짜는 문서 끝까지 일관되게 유지

**학교 가정통신문 상단 구조**
- 제목 영역 위 또는 옆에 학교 문서 정보 배치
  - 학교명 또는 교표 자리
  - 가정통신문 제목
  - 담당 부서
  - 발송 일자
  - 문의처
  - 홈페이지/주소는 입력된 경우만 사용
- 입력되지 않은 학교 기본값을 임의로 만들지 않음
- 예시 담당자명 `홍길동`, 계정 학교 주소 등 임시값 사용 금지

**핵심 정보 우선 배치 (역피라미드 구조)**
1. **첫 문단**: 무엇을, 언제 (핵심 사항)
2. **둘째 문단**: 왜, 어떻게 (배경 설명)
3. **셋째 문단**: 세부 안내 (준비물, 유의 사항 등)
4. **마지막**: 회신란/동의서 (해당 시)

**회신란 유형별 구성**
- 확인서: 안내 확인 여부, 보호자 성명, 보호자 서명
- 동의서: 동의/비동의 체크, 학생 정보, 보호자 서명
- 설문형: 선택 문항, 서술 의견, 제출 기한
- 실천 기록형: 실천 여부, 실천 주제, 느낀 점, 보호자 의견
- 문서 목적과 무관한 고정 항목 금지
  - 체험학습이 아닌데 `알레르기`, `비상 연락처`를 넣지 않음
  - 실천 활동 안내라면 `실천 여부`, `느낀 점`, `보호자 의견` 우선

**필수 포함 정보 체크리스트**
- [ ] 행사/안내 명칭
- [ ] 일시 (날짜 + 요일 + 시간)
- [ ] 장소
- [ ] 대상 학년/반
- [ ] 준비물 또는 유의 사항
- [ ] 회신 마감일 (해당 시)
- [ ] 담당 교사 연락처

## 4. AI 생성 텍스트 주의사항

### 4.1 과도한 수식어 지양

AI는 `매우`, `다양한`, `효과적인`, `중요한`, `적극적인` 등을 남용하는 경향이 있습니다. 다음 규칙을 적용하세요.

**삭제 대상 수식어 목록**
- `매우`, `상당히`, `굉장히` → 삭제하거나 구체적 수치로 대체
- `다양한` → 구체적 나열로 대체 (예: `다양한 활동` → `토의, 발표, 글쓰기 활동`)
- `효과적인`, `효율적인` → 삭제하거나 근거 제시
- `적극적인/적극적으로` → `적극` 또는 삭제
- `기본적인/기본적으로` → `기본` 또는 삭제
- `실질적인/실질적으로` → 삭제 (의미 기여 없음)

### 4.2 반복 표현 주의

**동일 문서 내 반복 패턴 회피**
- 같은 접속사 3회 이상 연속 사용 금지 (`또한`, `그리고`, `이를 통해`)
- 동일 문장 구조 반복 금지 (`~한다. ~한다. ~한다.`)
- 단락 시작어 다양화 (매 단락 `~은/는`으로 시작하지 않기)

**AI 특유 반복 패턴 (반드시 교정)**
| AI가 반복하는 표현 | 교정 방향 |
|-------------------|----------|
| 이를 통해 ~할 수 있습니다 | 구체적 결과 직접 서술 |
| ~하는 것이 중요합니다 | ~필요함 / ~권장 |
| ~에 대해 알아보겠습니다 | 삭제 (교육 문서에 불필요) |
| 다양한 ~을 활용하여 | 구체적 항목 나열 |
| ~라고 할 수 있습니다 | 단정형으로 전환 |

### 4.3 구체적 수치/사례 포함 권장

추상적 서술 대신 **구체적 수치, 기준, 사례**를 포함합니다.

| 추상적 서술 | 구체적 서술 |
|------------|-----------|
| 충분한 시간 제공 | 15분 이상 제공 |
| 여러 번 연습 | 3회 이상 반복 연습 |
| 적절한 분량 | A4 1매 이내 |
| 소규모 모둠 구성 | 4인 1모둠 구성 |
| 다수의 학생이 참여 | 학급 인원의 80% 이상 참여 |

### 4.4 문체 일관성 체크리스트

AI 생성 텍스트를 최종 검수할 때 다음을 확인합니다.

- [ ] 한 문서 내 종결 어미 통일 (개조식이면 전부 개조식)
- [ ] `-의` 조사 연쇄 2회 이상 없음
- [ ] `-적` 접미사 문서 전체에서 3회 이하
- [ ] 동일 접속사 연속 3회 미만
- [ ] 한 문장 46자 초과 항목 없음
- [ ] 추상 수식어(`다양한`, `효과적인`) 사용 시 구체적 보충 있음
- [ ] 학부모 대상 문서에 전문 용어 미설명 노출 없음
- [ ] 평가 문항 선지 길이 균등, 문법 구조 통일

## 5. 문서 유형별 종결 어미 요약

| 문서 유형 | 종결 어미 | 예시 |
|-----------|----------|------|
| 학습지 지시문 | 명령형 (`-하시오`, `-해 보자`) | 밑줄 친 부분의 의미를 쓰시오. |
| 평가지 발문 | 의문형 (`-는?`, `-인가?`) | 위 글의 주제로 적절한 것은? |
| 공문 본문 | 경어 개조식 (`-바랍니다`, `-알려 드립니다`) | 협조하여 주시기 바랍니다. |
| 수업 계획서 | 명사형 (`-함`, `-임`, `-실시`) | 학습 목표 확인 및 동기 유발 실시 |
| 가정통신문 | 경어 서술형 (`-습니다`, `-바랍니다`) | 자녀의 준비물을 확인해 주시기 바랍니다. |
| 교사용 메모 | 해체 (`-함`, `-할 것`) | 모둠 편성 사전 확인할 것 |
