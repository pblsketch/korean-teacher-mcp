# 목적별 루틴 선택 매트릭스 (Selection Matrix)

> korean-teacher-mcp | teacher-thinking-routines 참조 가이드

사용자의 활용 목적에 따라 어떤 분류의 사고 루틴을 우선 선택해야 하는지 매핑하는 가이드입니다. 3개의 루틴을 추천할 때 **서로 다른 사고 유형**을 섞어 다양성을 확보하는 것이 원칙입니다.

## 목적 → Project Zero 분류 매핑

| 활용 목적 | 우선 분류 | 대표 루틴 |
|-----------|-----------|-----------|
| 관계맺기·도입 | Introducing Ideas | See Think Wonder, Chalk Talk, Think Pair Share |
| 깊이 탐구하기 | Exploring Ideas | What Makes You Say That?, Claim Support Question, Circle of Viewpoints |
| 종합·조직하기 | Synthesizing and Organizing | Headlines, CSI (Color Symbol Image), Sentence Phrase Word |
| 조사·탐구하기 | Digging Deeper | Tug of War, The 4 C's, Peeling the Fruit |
| 성찰·메타인지 | Reflection/Metacognition | I Used to Think Now I Think, Compass Points, Connect Extend Challenge |

## 다양성 보장 조합 예시

추천 3개 루틴은 서로 다른 사고 유형을 포함해야 합니다.

### 조합 1: 단원 도입 → 본 학습 → 정리
1. **See Think Wonder** (관찰&질문) — 도입
2. **Claim Support Question** (추론) — 본 학습
3. **I Used to Think Now I Think** (성찰) — 정리

### 조합 2: 논쟁적 주제 탐구
1. **Circle of Viewpoints** (다관점) — 관점 수집
2. **Tug of War** (논쟁) — 입장 비교
3. **Headlines** (종합) — 결론 요약

### 조합 3: 개념 형성 과정
1. **Chalk Talk** (사전 지식) — 개념 활성화
2. **Connect Extend Challenge** (연결·확장) — 새 지식 통합
3. **CSI (Color Symbol Image)** (시각화) — 개념 정리

## 선택 시 추가 체크리스트

- [ ] 사용자가 명시한 **활용 목적**(도입/탐구/성찰 등)과 일치하는가?
- [ ] **학교급**에 맞는 복잡도인가? (저학년일수록 짧고 시각적)
- [ ] **교과목** 특성과 어울리는가?
- [ ] 3개 루틴이 **서로 다른 사고 유형**을 커버하는가?
- [ ] 수업 **시간 배분**을 고려했는가? (한 차시에 3개 루틴은 과다)
- [ ] 교사의 **선행 경험 유무**를 고려했는가? (초보 교사는 단순한 루틴부터)

## 주의: 루틴 선택이 아닌 경우

사용자의 요청이 아래에 해당하면 루틴 선택이 아니라 상위 설계가 먼저입니다.

- "수업 전체를 설계해줘" → `teacher-lesson-planning`
- "평가 문항을 만들어줘" → `teacher-assessment` / `teacher-quiz`
- "프로젝트 수업 주제" → `teacher-pbl`
- "학습지 형식" → `teacher-instructional-materials`
