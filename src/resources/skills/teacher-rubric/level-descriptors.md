# 수준 기술어 작성법

> korean-teacher-mcp | teacher-rubric 참조 가이드

수행 수준 기술(Performance Level Descriptions) 작성 원칙과 실제 기술어 작성 가이드입니다.

## 수행 수준 기술 작성 원칙

- **서술적 표현**: 작품에서 관찰되는 것을 기술 (평가적 판단 용어 회피)
- **명료성**: 학생과 교사 모두 이해 가능한 설명
- **전체 범위 포괄**: 각 준거의 질적 연속체 전체를 설명
- **수준 간 구별**: 각 수준이 명확히 구분되어 작품 분류가 가능함
- **목표 수준 중심**: 기대하는 수준이 적절한 위치에 배치됨
- **평행 구조**: 같은 준거의 각 수준이 동일한 측면을 다른 질적 수준으로 기술

## 평행 구조 작성법

각 준거의 수준 기술은 **같은 측면을 다른 질적 수준으로** 기술해야 합니다.

예시(주장의 명확성):
- 최상위 수준: "주장이 명확하고 일관되게 제시되며, 글 전체에서 독자가 주장을 쉽게 파악할 수 있다"
- 상위 수준: "주장이 대체로 명확하게 제시되며, 글의 대부분에서 주장을 파악할 수 있다"
- 중간 수준: "주장이 제시되지만 일부 모호한 부분이 있어 독자가 추론해야 한다"
- 하위 수준: "주장이 불분명하거나 일관되지 않아 파악이 어렵다"

## 평가적 판단 용어를 서술어로 변환

| 평가적 표현(피해야 함) | 서술적 표현(권장) |
|-----------------------|------------------|
| "우수함" | "근거가 여러 출처에서 수집되어 서로 뒷받침되며, 반대 입장도 검토된다" |
| "보통" | "주요 근거 2-3개가 제시되나 출처의 다양성은 제한적이다" |
| "미흡함" | "근거가 1개 이하 또는 주장과 직접 연결되지 않는다" |

## 수준 수 선택 가이드

- **3단계**: 기초·표준·심화. 형성평가·자기점검용으로 간결
- **4단계**: 노력 필요·기본·능숙·우수. 총괄평가의 표준
- **5단계**: 세밀한 변별이 필요한 대규모 평가에 적합

수준 수는 사용자가 요청한 개수(3~5단계)에 맞춰 작성합니다. 준거 수는 **항상 5개**를 작성하며, 각 준거는 성취기준의 핵심 문장을 근거로 도출하고 준거 설명(rationale)에 어떤 성취기준 문장을 반영했는지 명시합니다.

## 학생 친화적 언어 원칙

- 전문 용어보다 학생이 이해 가능한 풀어쓴 표현 사용
- 판단 대신 관찰: "잘했다" 대신 "~이 나타난다"
- 명확하고 구체적으로 작성하되 단순 계수(개수 기준)는 피함
