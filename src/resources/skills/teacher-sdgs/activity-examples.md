# SDGs 수업 활동 사례

> korean-teacher-mcp | teacher-sdgs 참조 가이드

학습자가 실제로 실행 가능하고 의미 있는 PBL(프로젝트 기반 학습) 주제를 설계할 때 참고할 수 있는 활동 관점입니다.

## 활동 설계 기준

- **실현가능성**: 학교 현장에서 실제 수행 가능한 규모, 학습자의 능력과 자원 범위 내 활동
- **학습자 주도성**: 학습자의 주도성과 선택권 보장, 흥미/관심사 반영
- **교육과정 연계**: 성취기준을 핵심 학습 목표로 반영, 교과 역량 및 핵심 개념과 융합
- **SDGs 통합**: 2-3개 연계 목표를 통합하여 다층적 접근

## 활동 유형 관점

PBL 주제를 SDGs와 연계할 때 다음과 같은 활동 유형을 고려합니다:

- **조사·탐구형**: 지역사회 또는 글로벌 이슈에 대한 자료 수집·분석
- **실천·참여형**: 학교·지역사회에서 실제 행동으로 옮기는 캠페인·프로젝트
- **창작·표현형**: 영상, 포스터, 연극 등 산출물 제작을 통한 메시지 전달
- **협업·토론형**: 모둠 토론, 롤플레이, 글로벌 교류 프로젝트
- **융합·통합형**: 여러 교과의 관점을 통합한 다교과 프로젝트

## 교사 컨설팅 관점

- 교사가 입력한 교육 맥락(학교급, 학년, 교과, 성취기준 등)을 먼저 분석합니다.
- 학습자 흥미/관심사가 명시되면 활동 설계에 직접 반영합니다.
- 자원(시간, 예산, 도구) 제약이 있으면 축소형으로 단계 조정합니다.
- JSON 응답에는 제안 주제별로 연계된 SDGs 번호, 교과 연결 근거, 기대 산출물을 포함하도록 안내합니다.

## 응답 형식

반드시 JSON 형식으로만 응답하며, 추가 설명이나 마크다운 없이 순수 JSON만 출력합니다. 구체 스키마는 `generate_pbl` 도구가 반환하는 구조를 따릅니다.
