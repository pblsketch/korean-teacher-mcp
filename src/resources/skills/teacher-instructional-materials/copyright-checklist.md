# 저작권 및 품질 체크리스트

> korean-teacher-mcp | teacher-instructional-materials 참조 가이드

수업 자료 제작 시 저작권·출처·품질 확인을 위한 체크리스트입니다.

## 저작권 기본 원칙

- 학교 수업 목적 이용은 저작권법 제25조에 따라 상당 부분 허용되지만, 공개·배포·출판 시에는 별도 허락 필요
- 교과서 원문 인용 시 출판사·단원·쪽수 표기
- 외부 텍스트(뉴스 기사·문학 작품) 인용 시 출처 명시 및 최소 인용 원칙 준수
- 이미지·도표는 공공누리·CCL 조건 확인 후 사용

## 자료 제작 전 확인 사항

- [ ] 원본 텍스트의 저작권자·출처를 확인했는가?
- [ ] 재구성 범위가 "수업 목적 이용" 한도를 넘지 않는가?
- [ ] 학생 이름·개인정보가 포함되지 않았는가?
- [ ] 교과서 지문 전면 재구성 시 "○○ 출판사 ○○교과서 ○단원 기반" 표시를 했는가?

## 읽기 자료 수준 조정 시 유지 사항

- 핵심 개념 동일 유지
- 내용 정확성(사실·수치) 보존
- 원본의 학술적·윤리적 관점 왜곡 금지
- 수준 조정 시에도 핵심 어휘 5-7개 유지

## 이미지·도표 사용 체크

- [ ] 출처가 공공누리·CCL·교과서 본문 중 하나인가?
- [ ] 학생 작품을 활용할 경우 학생·보호자 동의를 받았는가?
- [ ] AI 생성 이미지의 경우 생성 도구·프롬프트를 내부 기록에 남겼는가?

## 한글 파일(HWPX) 호환성 체크

- [ ] 표 폭이 42520 이하인가?
- [ ] cellMargin이 좌/우 510, 상/하 141 기준인가?
- [ ] 중첩 표·래퍼 표가 없는가?
- [ ] 단락-표 교차 구조를 유지하는가?
- [ ] 특수 기호가 한글 표준 글꼴(함초롬바탕)에서 표현 가능한가?

## 학생 안전(실험·활동) 체크

- [ ] 실험 가이드에 안전 수칙이 단계별 절차 앞에 배치되어 있는가?
- [ ] 위험 시약·도구 취급 주의 표시가 명확한가?
- [ ] 사고 발생 시 대처 방법이 안내되어 있는가?
- [ ] 가정 추가 탐구 제안 시 안전 한도 내 활동인가?

## 포용성·UDL 체크

- [ ] 다양한 읽기 수준을 고려했는가?
- [ ] 시각·청각 자료를 함께 제공했는가?
- [ ] 특정 성별·문화·경제 배경 편향이 없는가?
- [ ] 학생이 자기 수준을 선택할 수 있는 옵션이 있는가?

## 배포 전 최종 점검

- [ ] 파일명·제목에 학년·교과·단원·차시가 드러나는가?
- [ ] 인쇄 시 A4 한 면에 내용이 적절히 배치되는가?
- [ ] 자기 점검·피드백 공간이 확보되어 있는가?
- [ ] 교사용 안내(정답·진행 시나리오)는 별도 파일로 분리했는가?
