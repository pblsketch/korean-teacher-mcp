# 문항 유형별 설계 가이드

> korean-teacher-mcp | teacher-quiz 참조 가이드

PBL(프로젝트 기반 학습) 환경에서 사용할 5종 형성평가 문항을 설계할 때 적용하는 유형별 원칙입니다.

## 1단계: 콘텐츠 분석

제공된 텍스트에서 다음을 추출합니다.

- **핵심 개념**: 정의, 특징, 구성요소
- **사실 정보**: 숫자, 날짜, 이름, 장소
- **관계**: 원인-결과, 비교-대조, 순서-절차
- **주장/논점**: 저자의 견해, 결론

## 2단계: 문항 유형별 설계

### 선다형 (Multiple Choice)

- **정답**: 콘텐츠에 명시된 정보
- **오답 구성 전략**:
  * 콘텐츠 내 다른 개념과 혼동 유도
  * 비슷하지만 틀린 표현 사용
  * 일반적인 오개념 반영
  * 부분적으로만 맞는 답 제시
- **난이도 조절**:
  * 쉬움: 직접 언급된 사실 확인
  * 보통: 개념 이해 및 적용
  * 어려움: 여러 정보 종합, 분석적 사고

### 힌지 질문 (Hinge Question)

- **4지선다 고정**
- 각 오답은 특정 오개념을 진단
- **misconceptionMap 필수**: 오답 인덱스(문자열)를 키로 하여 오개념과 재교육 전략 포함
- **teacherGuidance 필수**: 활용 시점과 진행 판단 기준

### 진위형 (True/False)

- 콘텐츠 문장을 변형하여 진술문 작성
- 거짓 진술: 핵심 단어나 관계를 미묘하게 변경

### 단답형 (Short Answer)

- 콘텐츠에 명확히 등장하는 용어/개념을 정답으로
- 다양한 정답 표현을 expectedAnswerKeywords에 포함

### 서술형 (Essay)

- 콘텐츠 내용을 바탕으로 설명, 비교, 적용하는 질문
- **modelAnswer 필수**: 모범 답안 예시 제공
- **essayRubric 필수**: 채점 루브릭 (평가 준거, 수준별 기준, 총점)

## 3단계: 품질 검증

각 문항에 대해 다음을 확인합니다.

1. **정답 검증**: 콘텐츠에서 정답의 근거를 찾을 수 있는가?
2. **오답 검증**: 오답이 그럴듯하면서도 명확히 틀린가?
3. **난이도 적절성**: 요청된 난이도에 맞는가?
4. **교육적 가치**: 단순 암기가 아닌 이해를 평가하는가?
