# 평가 프롬프트 템플릿 모음

> korean-teacher-mcp | teacher-assessment 참조 가이드

평가 유형별 기본 템플릿과 예시 프롬프트입니다.

## 2.1 퀴즈 및 문항 생성 기본 템플릿

```
당신은 [교과] 평가 문항 개발 전문가입니다. [학년] 학생들의 [주제/개념] 이해도를 평가하는 퀴즈를 만들어주세요.

**성취기준:** [관련 성취기준]
**평가 목적:** [형성평가/총괄평가/진단평가]
**문항 수:** [원하는 문항 수, 기본 20문항]
**문항 유형:** [선택형/보기형/서술형/혼합]

다음을 포함해주세요:
- 문항별 평가 요소
- 정답 및 해설
- 예상 오답과 오개념 분석
```

### 예시: 개념 이해 형성평가 퀴즈

```
당신은 중학교 과학 평가 문항 개발 전문가입니다. 중학교 2학년 학생들의 세포 분열 개념 이해를 평가하는 형성평가 퀴즈를 만들어주세요.

**성취기준:** [9과05-02] 세포 분열을 체세포 분열과 생식세포 분열로 구분하고, 염색체의 변화를 설명할 수 있다.
**평가 목적:** 수업 중 형성평가 (학습 점검)
**문항 수:** 5문항
**문항 유형:** 선택형 3문항 + 단답형 2문항

다음을 포함해주세요:
- 문항별 평가 요소 (지식/이해/적용 구분)
- 정답 및 간단한 해설
- 학생들이 자주 보이는 오개념 분석
- Bloom 인지 수준 표시
```

## 2.2 수행과제 설계 기본 템플릿

```
당신은 [교과] 수행평가 설계 전문가입니다. [학년] 학생들을 위한 [주제] 관련 수행과제를 설계해주세요.

**성취기준:** [관련 성취기준]
**과제 유형:** [프로젝트/발표/포트폴리오/실험/글쓰기]
**수행 기간:** [기간]
**평가 방식:** [개별/모둠]

다음을 포함해주세요:
- 학생용 과제 안내문
- 평가 기준표 (루브릭)
- 과정 평가 체크포인트
- 우수 예시 설명
```

### 예시: 논쟁적 주제 토론 수행평가

```
당신은 고등학교 국어과 수행평가 설계 전문가입니다. 고등학교 1학년 학생들을 위한 '논증적 말하기' 수행과제를 설계해주세요.

**성취기준:**
- [10국01-04] 협상을 통해 의견과 관계를 조정한다.
- [10국01-05] 가치 있는 논제를 선정하여 논증을 구성하고 토론에 참여한다.

**과제 유형:** 모둠 토론 + 개별 성찰 보고서
**수행 기간:** 2주 (준비 1주 + 수행 1주)
**평가 방식:** 모둠 평가(50%) + 개별 평가(50%)

다음을 포함해주세요:
- 학생용 과제 안내문 (A4 1장 분량)
- 토론 평가 루브릭 (4단계)
- 개인 성찰 보고서 평가 기준
- 모둠별 역할 분담 가이드
- 과정 평가 체크포인트 3개
```

## 2.3 루브릭 개발 기본 템플릿

```
당신은 루브릭 설계 전문가입니다. [과제명]의 평가를 위한 분석적 루브릭을 개발해주세요.

**평가 대상:** [학년/교과]
**평가 과제:** [과제 설명]
**평가 영역:** [평가할 영역/기준]
**수준 단계:** [3단계/4단계/5단계]

다음을 포함해주세요:
- 영역별 상세 평가 기준
- 수준별 구체적 행동 지표
- 각 수준의 예시 설명
- 학생 자기평가용 간소화 버전
```

### 예시: 프로젝트 기반 학습 루브릭

```
당신은 교육 평가 루브릭 설계 전문가입니다. 중학교 과학 '환경 문제 해결 프로젝트'의 평가를 위한 분석적 루브릭을 개발해주세요.

**평가 대상:** 중학교 3학년
**평가 과제:** 지역 환경 문제 분석 및 해결 방안 제안 발표
**평가 영역:**
1. 과학적 탐구 능력 (자료 수집, 분석)
2. 문제 해결력 (해결 방안의 창의성, 실현가능성)
3. 의사소통 (발표 구성, 전달력)
4. 협력 (모둠 기여도, 협력적 태도)

**수준 단계:** 4단계 (매우 우수-우수-보통-노력 필요)
```

## 2.4 기존 평가 변형 기본 템플릿

```
당신은 [학교급] [교과] 평가 전문가입니다. 기존 평가를 동일한 학습 목표를 유지하면서 대안적 버전으로 재구성해주세요.

**원본 평가:** [첨부 또는 설명]
**변형 목적:** [재시험/보충 학습/수준별 평가]
**유지 사항:** [동일하게 유지할 요소]
**변경 사항:** [변경이 필요한 요소]
```

### 예시: 수준별 평가 변형

```
당신은 중학교 수학 평가 전문가입니다. 다음 일차방정식 시험을 동일한 학습 목표를 유지하면서 3가지 수준의 버전으로 재구성해주세요.

**변경 사항:**
- 기초 수준: 단순 계산, 직접적 적용
- 표준 수준: 원본과 유사
- 심화 수준: 복합 문제, 실생활 적용

각 버전에 대해:
- 난이도 조정 원리 설명
- 문항별 변경 내용 명시
- 채점 기준 통일 방안
```

## 2.5 데이터 기반 수업 개선 예시

```
당신은 교육 데이터 분석 전문가입니다. 다음 형성평가 결과를 분석하고 후속 수업 제안을 해주세요.

**평가 정보:** 중학교 2학년 수학, 연립방정식, 10문항, 28명
**결과 데이터:** [문항별 정답률 또는 학생별 점수 첨부]

다음을 분석해주세요:
1. 전체 학습 도달도 분석
2. 문항별 정답률과 오류 패턴
3. 공통 오개념 식별
4. 재지도가 필요한 핵심 개념
5. 수준별 소그룹 구성 제안
6. 후속 수업 활동 3가지 제안
```

## 2.6 사례 기반 평가 설계 예시

```
당신은 대학교 교육학과 평가 설계 전문가입니다. 교육심리학 입문 수업을 위한 사례 기반 평가를 설계해주세요.

**평가 주제:** 학습 동기 이론의 실제 적용
**대상:** 교육학과 2학년 학생
**평가 형식:** 사례 분석 + 서술형 응답

다음을 포함해주세요:
1. 실제 교실 상황을 반영한 사례 시나리오 (500자 내외)
2. 분석 질문 3개 (이론 적용, 대안 제안, 자기 성찰)
3. 루브릭 (증거 활용, 개념 정확성 기준 포함)
4. 학생 자기평가용 안내문
```
