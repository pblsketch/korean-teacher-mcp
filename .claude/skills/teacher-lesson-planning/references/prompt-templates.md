# 수업 설계 프롬프트 예시

> korean-teacher-mcp | teacher-lesson-planning 참조 가이드

실제 교실 맥락에서 바로 활용 가능한 예시 프롬프트 모음입니다.

## 📌 5E 모형 기반 과학 수업 설계

```
당신은 중학교 과학 교육과정 설계 전문가입니다. 중학교 2학년 학생들을 위한 2차시 수업을 5E 모형으로 설계해주세요.

**학습 주제:** DNA 구조와 기능
**성취기준:** [9과08-01] 유전 물질이 DNA임을 알고, DNA의 구조적 특성을 이해한다.
**수업 모형:** 5E 탐구학습 모형

각 단계(참여-탐색-설명-정교화-평가)별로 다음을 포함해주세요:
- 핵심 활동 설명
- 발문 예시
- 학생 참여 방법
- 소요 시간

총 수업 시간은 100분(50분 x 2차시)입니다.
```

## 📌 백워드 설계 기반 국어 단원 설계

```
당신은 고등학교 국어과 단원 설계 전문가입니다. 4주(12차시) 동안의 '비평적 읽기와 쓰기' 단원을 백워드 설계 원리로 개발해주세요.

**대상:** 고등학교 1학년, 기초 학력이 다양한 일반계 고등학교
**관련 성취기준:**
- [10국02-03] 다양한 관점과 방법으로 글을 읽는다.
- [10국03-04] 글의 내용이나 형식에 대한 비평적 의견을 제시한다.

**핵심 질문:** "비평적 읽기는 어떻게 우리의 사고를 확장하는가?"

다음을 포함해주세요:
1. 단원 학습 목표 (지식, 기능, 태도)
2. 12차시 주차별 학습 내용 및 활동
3. 수행평가 과제 및 루브릭
4. 차시별 형성평가 방법
5. 교차 교과 연계 가능성
```

## 📌 환경과학 + 국어 융합 프로젝트

```
당신은 교과 연계 프로젝트 기획 전문가입니다. 중학교 3학년 학생들을 위한 2주 프로젝트를 개발해주세요.

**연계 교과:** 과학(환경) + 국어(설득적 글쓰기)
**프로젝트 주제:** 우리 지역 환경 문제 해결을 위한 정책 제안서 작성
**최종 산출물:** 지역 환경 문제 분석 보고서 + 정책 제안 프레젠테이션

다음을 포함해주세요:
- 각 교과별 관련 성취기준 명시
- 일자별 활동 계획 (과학 수업 + 국어 수업 연계)
- 모둠 협력 활동 설계
- 과정 평가와 결과 평가 루브릭
- 학생 역할 분담 가이드
```

## 📌 AI 수업 활용 교사 연수

```
당신은 교사 전문성 개발 설계 전문가입니다. 중학교 교사들을 위한 90분 AI 활용 수업 연수를 기획해주세요.

**연수 주제:** Gemini를 활용한 수업 설계와 학생 학습 지원
**연수 목표:**
- 효과적인 프롬프트 작성 능력 습득
- 교과별 AI 활용 사례 이해
- 실제 수업 적용 계획 수립

**연수 형태:** 실습 중심 워크숍

다음을 포함해주세요:
- 90분 세부 일정 (15분 단위)
- 참여자 실습 활동 3개 이상
- 제공할 자료 목록
- 연수 후 실천 과제
- 후속 학습 공동체 운영 방안
```

## 📌 개념 연결 학습 시퀀스

```
당신은 과학 교육과정 설계 전문가입니다. 중학교 2학년 물리 영역 '힘과 운동' 단원의 5일 학습 시퀀스를 설계해주세요.

**조건:**
- 학생들의 사전 지식이 다양함 (일부는 기초 개념 부족)
- 매일 핵심 개념 간 연결이 명확해야 함
- 탐구 활동, 협력 학습, 개별 활동이 균형있게 포함

다음 형식으로 제시해주세요:
| 일차 | 핵심 개념 | 주요 활동 | 형성평가 | 다음 차시 연결 |
|------|----------|----------|----------|---------------|
```
