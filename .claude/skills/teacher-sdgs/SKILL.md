# 교과서 SDGs (Korean Teacher SDGs)

> korean-teacher-mcp | UN 지속가능발전목표 교육 연계 전문 스킬

참조 가이드:
- `references/sdgs-curriculum-map.md` — 17개 목표 개요와 교과 연계 관점
- `references/activity-examples.md` — 수업 활동 유형과 교사 컨설팅 관점

## 트리거 키워드

SDGs, 지속가능발전목표, 지속가능발전, UN 목표, 글로벌 이슈, 기후 변화 프로젝트, 환경 PBL, 사회 참여 프로젝트, 세계시민교육

## 워크플로우

### 1단계: 교육 맥락 분석
- 학교급, 학년, 교과, 성취기준, 학습자 흥미/관심사를 확인
- `search_content`로 성취기준 원문과 교과 내용을 확보

### 2단계: 연계 목표 선정
- 17개 SDGs 중 맥락과 가장 연관성 높은 1개를 중심으로 2-3개 목표를 통합
- 교과 역량·핵심 개념과의 자연스러운 융합 지점 도출

### 3단계: 주제 설계
- 실현가능성(자원·시간·학습자 능력)과 학습자 주도성 원칙 적용
- 학습자 흥미와 주도성·선택권을 반영한 프로젝트 얼개 구성

### 4단계: 출력
- `generate_pbl`로 JSON 주제 목록 생성 (마크다운·설명 없이 순수 JSON)
- 한글 문서 필요 시 `export_hwpx`

## MCP 도구 연동

- `search_content` — 성취기준과 교과 내용 확보로 연계 근거 고정
- `generate_pbl` — SDGs 연계 PBL 주제를 JSON으로 생성
- `export_hwpx` — 한글 문서로 내보내기 (요청 시)

## 산출물

- SDGs 2-3개와 통합된 PBL 주제(순수 JSON)
- 주제별 연계 목표 번호·교과 연결 근거·기대 산출물
- 학습자 주도성·선택권이 반영된 활동 얼개

## 사용 예시

- "중3 사회 수업에 쓸 SDGs 연계 PBL 주제 추천해줘"
- "고1 과학 기후 변화 단원에 SDG 13, 14 연계한 프로젝트 제안해줘"
- "초6 도덕 성취기준에 맞는 SDGs 주제 3개 뽑아줘"
- "지역사회 참여형 SDGs 프로젝트 설계해줘"
- "학생 흥미가 동물 보호인데 SDGs 목표와 연결되는 주제 있어?"

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 주제가 학교 현장에서 실행 불가능해 보인다 | 실현가능성 원칙으로 자원·시간·학습자 능력 범위 내로 축소 |
| SDGs가 교과 내용과 분리된 느낌이다 | 성취기준을 핵심 학습 목표로 먼저 고정 후 2-3개 목표 통합 |
| 학습자 주도성이 낮다 | 학습자 흥미·선택권을 설계에 직접 반영해 다시 생성 |

## 이 스킬을 사용하지 말아야 할 때

- **일반 PBL 주제 설계(SDGs 비연계)** → `teacher-pbl` 스킬 사용
- **수업 전체·차시 설계** → `teacher-lesson-planning` 스킬 사용
- **평가 계획·루브릭·문항 제작** → `teacher-assessment` 스킬 사용
