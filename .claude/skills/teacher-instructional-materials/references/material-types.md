# 수업 자료 유형별 가이드

> korean-teacher-mcp | teacher-instructional-materials 참조 가이드

학습지, 읽기 자료, 실험 가이드, 토론 자료, 스킬 연습 활동 개발을 위한 유형별 프롬프트 가이드입니다.

## 수업 자료 프롬프트의 PARTS 적용

| 요소 | 수업 자료 맥락에서의 적용 |
|------|--------------------------|
| **P: Persona** | 학습 자료 개발자, 실험 설계 전문가, 콘텐츠 편집자 |
| **A: Act** | 생성하다, 수정하다, 수준 조정하다, 요약하다 |
| **R: Recipient** | 특정 학년/읽기 수준/학습 특성의 학생 |
| **T: Theme** | 학습 주제, 교과 내용, 활동 목적 |
| **S: Structure** | 학습지 형식, 실험 절차, 토론 가이드 형식 |

## 3.1 읽기 자료 수준 조정 (Re-level Reading)

### 기본 템플릿

```
당신은 교과 읽기 자료 전문가입니다. 다음 텍스트를 [대상 학생]에게 적합한 수준으로 재구성해주세요.

**원본 텍스트:** [첨부 또는 입력]
**대상 학년:** [학년]
**수준 조정 유형:** [쉽게/어렵게/다중 수준]
**유지 사항:** [핵심 개념, 내용 정확성]

다음을 포함해주세요:
- 조정된 텍스트
- 어휘 설명 (필요시)
- 이해 확인 질문 2-3개
```

### 예시: 다중 수준 읽기 자료 개발

중학교 1학년 대상 '광합성' 지문을 3가지 수준으로:
- 기초 수준: 초등 5-6학년 수준, 짧은 문장, 그림 설명 포함 제안
- 표준 수준: 중학교 1학년 수준, 교과서와 유사한 난이도
- 심화 수준: 중학교 2-3학년 수준, 추가 개념과 연결

**활용 팁**
- "학생 친화적 언어"를 강조하면 교과서 문체보다 친근한 톤이 됩니다
- "시각적 단서 제안"을 요청하면 그림, 도표 삽입 위치가 표시됩니다

## 3.2 학생용 학습지 (Student Handout)

### 기본 템플릿

```
당신은 [교과] 학습지 개발 전문가입니다. [학년] 학생들을 위한 [주제/기능] 학습지를 만들어주세요.

**학습 목표:** [구체적인 학습 목표]
**학습지 유형:** [개념 정리/연습 문제/탐구 활동/자기 점검]
**분량:** [A4 몇 장]
**소요 시간:** [예상 소요 시간]

다음을 포함해주세요:
- 학생용 안내문
- 활동 내용
- 자기 점검 요소
```

### 학습지 기본 구성

```
┌─────────────────────────────────────┐
│ [교과] [단원명] 학습지              │
│ ○학년 ○반 ○번 이름: _________      │
├─────────────────────────────────────┤
│ 학습 목표                           │
│ • [목표 1]                          │
│ • [목표 2]                          │
├─────────────────────────────────────┤
│ 핵심 개념 정리                      │
│ [개념 설명 영역]                    │
├─────────────────────────────────────┤
│ 활동 1: [활동명]                    │
│ [활동 내용]                         │
├─────────────────────────────────────┤
│ 자기 점검                           │
│ □ [점검 항목 1]                     │
│ □ [점검 항목 2]                     │
├─────────────────────────────────────┤
│ 선생님 피드백                       │
│ [피드백 공간]                       │
└─────────────────────────────────────┘
```

## 3.3 실험/실습 가이드 (Lab Guide)

### 기본 템플릿

```
당신은 [교과] 실험 설계 전문가입니다. [학년] 학생들을 위한 [실험/실습명] 가이드를 만들어주세요.

**실험 목적:** [학습 목표와 연결된 실험 목적]
**소요 시간:** [예상 시간]
**준비물:** [필요한 재료/도구]

다음을 포함해주세요:
- 학생용 실험 안내서
- 안전 수칙
- 단계별 절차
- 관찰/기록 양식
- 분석 질문
```

**활용 팁**
- "탐구 중심"을 강조하면 결과 예측, 가설 설정 단계가 포함됩니다
- "오류 가능성"을 요청하면 흔한 실수와 해결책이 포함됩니다

## 3.4 토론 가이드 (Discussion Guide)

### 기본 템플릿

```
당신은 [교과] 토론 수업 설계 전문가입니다. [학년] 학생들을 위한 [주제] 토론 가이드를 만들어주세요.

**토론 유형:** [찬반 토론/세미나/하브루타/원탁 토론]
**주제/질문:** [토론 주제 또는 핵심 질문]
**소요 시간:** [예상 시간]
**참여 인원:** [모둠 구성]

다음을 포함해주세요:
- 배경 자료 요약
- 토론 질문 세트
- 역할 배분 가이드
- 토론 진행 절차
- 성찰 질문
```

## 3.5 기능 연습 활동 (Skill Practice)

단계별 연습 구성 예시(글쓰기):
- 1단계: 주장에 맞는 근거 선택하기
- 2단계: 근거를 주장과 연결하는 문장 쓰기
- 3단계: 짧은 주장 문단 완성하기

포함 요소: 기능 설명 / 모범 예시(좋은 vs 약한) / 단계별 연습 / 자기 점검 체크리스트 / 동료 피드백 양식

## 3.6 콘텐츠 개인화 (Content Personalization)

학생 관심 분야별로 같은 개념을 맥락화하여 재구성 (스포츠·게임·음악·환경 등).
각 버전 포함:
- 맥락화된 개념 설명 (200자)
- 관련 예제 문제 2개
- 실생활 연결 질문 1개

## 프롬프트 품질 체크리스트

- [ ] 대상 학년과 수준이 명시되어 있는가?
- [ ] 학습 목표가 명확한가?
- [ ] 자료 유형과 형식이 구체적인가?
- [ ] 분량과 소요 시간이 적절한가?
- [ ] 학생용 자기 점검 요소가 포함되어 있는가?
- [ ] 한글 파일 호환 형식이 요청되어 있는가?

## 후속 프롬프트 제안

1. **수준 조정**: "이 자료를 기초 수준 학생용으로 조정해주세요"
2. **평가 연계**: "이 활동의 평가 기준표를 만들어주세요"
3. **교사 가이드**: "이 자료 활용을 위한 교사용 안내서를 만들어주세요"
4. **디지털 변환**: "이 활동을 온라인 협업 도구용으로 변환해주세요"
