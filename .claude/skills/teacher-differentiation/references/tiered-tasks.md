# 수준별 과제 설계 (Tiered Tasks)

> korean-teacher-mcp | teacher-differentiation 참조 가이드

동일한 학습 목표 아래에서 학생의 준비도·수준에 따라 과제를 3단계로 설계하기 위한 템플릿과 예시, 개별화 수준 가이드를 제공합니다.

## 4.2 수준별 활동 설계 (Tiered Activities)

### 기본 템플릿

```
당신은 수준별 학습 설계 전문가입니다. [주제] 학습을 위한 3단계 수준별 활동을 설계해주세요.

**학습 목표:** [모든 수준에 공통된 학습 목표]
**대상:** [학년]
**활동 유형:** [탐구/연습/프로젝트/토론]

다음 수준별로 설계해주세요:
- 기초: [지원이 필요한 학생]
- 표준: [학년 수준에 도달한 학생]
- 심화: [추가 도전이 필요한 학생]

각 수준에 포함:
- 활동 설명
- 필요한 지원/도전 요소
- 예상 소요 시간
- 성공 기준
```

### 예시 — 수학 수준별 문제 해결 활동

```
당신은 수준별 수학 학습 설계 전문가입니다. '일차방정식의 활용' 학습을 위한 3단계 수준별 활동을 설계해주세요.

**학습 목표:** 실생활 상황을 일차방정식으로 모델링하고 해결할 수 있다
**대상:** 중학교 1학년
**활동 유형:** 문제 해결 + 모둠 협력

**기초 수준 (지원 필요):**
- 구조화된 문제 (식 세우기 틀 제공)
- 시각적 보조 자료
- 단계별 힌트 카드

**표준 수준 (학년 수준):**
- 다양한 맥락의 실생활 문제
- 자기 점검 체크리스트
- 동료 설명 활동

**심화 수준 (도전 필요):**
- 복잡한 조건의 문제
- 자신만의 문제 만들기
- 여러 가지 해결 전략 비교

각 수준에 포함해주세요:
- 구체적인 활동 설명 및 예시 문제
- 필요한 학습 지원 또는 도전 요소
- 예상 소요 시간
- 자기 평가 기준
```

## 학습 내용별 개별화 매트릭스

| 개별화 요소 | 기초 | 표준 | 심화 |
|------------|------|------|------|
| **복잡성** | 단순, 구조화 | 학년 수준 | 복잡, 다층적 |
| **추상성** | 구체적, 시각적 | 균형 | 추상적, 개념적 |
| **자원** | 지원 자료 다수 | 표준 자원 | 확장 자원 |
| **속도** | 충분한 시간 | 표준 시간 | 압축/가속 |
| **독립성** | 교사 지원 많음 | 적절한 지원 | 자기주도 |

## 수준별 과제 설계 점검 사항

### 모든 수준 공통
- [ ] 학습 목표가 동일하게 유지되는가?
- [ ] 결과물의 형식이 비교 가능한가?
- [ ] 평가 기준이 모든 수준에 적용 가능한가?

### 기초 수준
- [ ] 진입 장벽이 충분히 낮은가?
- [ ] 시각·청각 보조 자료가 포함되었는가?
- [ ] 단계가 작은 단위로 분할되었는가?
- [ ] 학생이 성공 경험을 가질 수 있는가?

### 표준 수준
- [ ] 학년 평균 학생이 75% 이상 도달 가능한가?
- [ ] 동료 협력 기회가 있는가?
- [ ] 자기 점검 체크리스트가 있는가?

### 심화 수준
- [ ] 단순한 양적 추가가 아닌 질적 도전인가?
- [ ] 학생 주도성·창의성이 발휘될 여지가 있는가?
- [ ] 추가 자원이나 사고 도구가 제공되는가?

## 수준별 과제 설계 시 자주 하는 실수

| 실수 | 교정 방향 |
|------|-----------|
| 기초 수준 = 양 줄이기 | 양이 아닌 구조·지원을 조정 |
| 심화 수준 = 양 늘리기 | 양이 아닌 복잡성·추상성·자기주도성 강화 |
| 수준별 학습 목표가 다르다 | 목표는 동일, 도달 경로만 차별화 |
| 학생을 고정 그룹으로 분류 | 활동·단원별 유연 배치, 라벨링 회피 |
| 평가가 수준별로 분리되어 비교 불가 | 공통 루브릭에 수준별 기대 수준 표기 |

## 활용 팁

- 수준별 그룹은 **고정이 아닌 유동적**으로 운영
- 학생이 자기 수준을 선택하는 옵션을 제공하면 자기조절 능력 향상
- 수준 명칭은 "기초/표준/심화" 외에 색깔·동물·기호로 중립화 가능
- 동일 차시에 모든 수준을 운영하기 어렵다면 시간차 도입 고려
