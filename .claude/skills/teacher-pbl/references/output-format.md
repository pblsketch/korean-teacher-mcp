# PBL 주제 추천 JSON 출력 형식

> korean-teacher-mcp | teacher-pbl 참조 가이드

프로젝트 주제 추천 응답은 반드시 아래 JSON 형식만 반환합니다.

## JSON 구조

```json
{
  "topics": [
    {
      "title": "주제 제목 (간결하고 명확하게)",
      "description": "주제에 대한 2-3문장 설명 (왜 중요한지, 무엇을 탐구할 수 있는지)"
    }
  ]
}
```

## JSON 출력 규칙 (중요)

- 반드시 **순수 JSON 문자열**만 반환하세요. 추가 설명, 인사말, 코드 블록(```)은 사용하지 마세요.
- JSON 객체는 `{"topics": [...]}` 한 개만 포함해야 합니다.
- 문자열 안에서 대괄호는 그대로 사용하세요. 예: `[12독작01-01]`
- 사용할 수 있는 이스케이프 문자는 표준 JSON만 허용됩니다 (`\"`, `\\`, `\n`, `\t`, `\b`, `\f`, `\r`, `\uXXXX`).
- 각 topic은 `title`과 `description`을 반드시 포함해야 하며, description은 2-3문장으로 작성합니다.
- topic 개수는 **3-5개 사이**로 제한합니다.

## 출력 전 체크

1. 순수 JSON만 반환했는가? (인사말·코드펜스 없음)
2. 최상위 키가 `topics` 하나뿐인가?
3. 각 topic에 `title`, `description`이 모두 있는가?
4. description이 2-3문장 분량인가?
5. topics 배열 길이가 3~5 범위인가?
